Course Overview

한 줄 요약

이 강의는 실무에 즉시 적용할 수 있는 핵심 개념과 실습을 다루는 1일 강의 과정입니다.

학습 목표

  • 핵심 개념을 설명할 수 있다
  • 실무 시나리오를 적용할 수 있다
  • 문제 해결 과정을 실행할 수 있다

대상자

관련 분야 1~3년 경력의 실무자. 기초 지식이 있으나 체계적인 이해가 필요한 사람.

사전 요구사항

  • Git 기본 조작 가능
  • 터미널 명령어 기본 지식
  • 사전 설치된 개발 환경
