# Celebrate

Use this skill when a task has been successfully completed.

## When to Use

- After successful deployment
- After resolving a difficult bug
- After completing a feature
- After passing all tests

## Output Pattern

```bash
# Example output
echo "🎉 Task Completed Successfully!"
echo "🚀 Code deployed and verified."
echo "✨ Great job, Team!"
```

## Variations

Adjust the message based on the achievement:

| Achievement | Message |
|-------------|---------|
| Feature complete | "🎉 Feature deployed!" |
| Bug fixed | "🐛 Bug squashed!" |
| Tests passing | "✅ All green!" |
| Deployment | "🚀 Ship it!" |
