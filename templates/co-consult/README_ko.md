# co-consult

> **Status**: Stable — v1.0.0

AI 기반 지식 업무 워크스페이스 템플릿. 구조화된 멀티 에이전트 아키텍처를 통해 문서 중심 프로젝트, 리서치, 크로스펑셔널 팀 협업에 최적화되어 있습니다.

## 개요

co-consult 템플릿은 리서치, 전략 수립, 클라이언트 산출물 작성을 위해 AI 어시스턴트를 활용하는 컨설팅 프로젝트를 위한 즉시 사용 가능한 워크스페이스를 제공합니다. Claude Code(`CLAUDE.md`)와 Gemini(`GEMINI.md`)를 구조화된 멀티 에이전트 컨설팅 팀과 결합하여, 모든 인게이지먼트에 첫날부터 일관된 명령, 에이전트 역할, 품질 관리를 제공합니다.

## 빠른 시작

> 워크스페이스 루트에서 새 프로젝트를 스캐폴딩하려면 다음을 실행하세요:
> `scripts/new-project.sh "project-name" --variant co-consult`

## 에이전트

| 에이전트 | 역할 |
|----------|------|
| **Engagement Leader** | 인게이지먼트 오케스트레이션, 클라이언트 인터페이스, 품질 게이트 관리 |
| **Change Management Partner** | 조직 변화 관리, 문화 전략, 이해관계자 정렬 |
| **Strategy Analyst** | 시장 분석, 경쟁 조사, 재무 모델링 |
| **Communications Lead** | 클라이언트 커뮤니케이션, 프레젠테이션, 전략 내러티브 |
| **Solutions Architect** | 기술 솔루션 설계, 시스템 아키텍처, 구현 로드맵 |
| **Delivery Manager** | 프로젝트 인도, 운영 조율, 이해관계자 관리 |
| **Technology Specialist** | M365 플랫폼, 워크플로우 자동화, 디지털 전환 지원 |
