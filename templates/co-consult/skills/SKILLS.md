# Skills Index — co-consult

This directory contains variant-specific skills for the `co-consult` template.

## Available Skills

| Skill | Directory | Purpose |
|-------|-----------|---------|
| Change Impact Assessment | `change-impact-assessment/` | Assess the impact of proposed changes on stakeholders and systems |
| Competitive Intelligence | `competitive-intelligence/` | Gather and analyze competitor data to inform consulting recommendations |
| Consulting Report Writing | `consulting-report-writing/` | Structure and write professional consulting deliverables |
| Executive Presentation | `executive-presentation/` | Create executive-level slide decks and briefings |
| Financial Modeling | `financial-modeling/` | Build financial models and quantitative analyses |
| Insight Synthesis | `insight-synthesis/` | Synthesize research and data into actionable insights |
| Narrative Framework | `narrative-framework/` | Develop compelling storylines for consulting engagements |
| Org Readiness Assessment | `org-readiness-assessment/` | Evaluate organizational readiness for change initiatives |
| Project Delivery | `project-delivery/` | Manage and track consulting project delivery milestones |
| Solution Design | `solution-design/` | Design end-to-end solutions for client problems |
| Stakeholder Alignment | `stakeholder-alignment/` | Facilitate stakeholder alignment workshops and sessions |
| Stakeholder Review Management | `stakeholder-review-management/` | Manage review cycles and stakeholder feedback loops |
| Technical Feasibility | `technical-feasibility/` | Assess technical feasibility of proposed solutions |
| k-dart | `k-dart/` | Query DART OpenAPI for Korean financial disclosures, company profiles, financial statements, and major event reports |
| Company Intelligence | `company-intelligence/` | Comprehensive company and corporate group intelligence gathering and analysis |

## Usage

Skills are invoked by the PM orchestrator or team members using the trigger phrases defined in each `SKILL.md` file.
