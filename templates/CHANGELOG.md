## [0.4.0] - 2026-05-26
### Added
- Multi-agent `/meeting` command with inline role-play orchestration
- Silent mode (default) and `--dialogue` opt-in for meeting transcripts
- Variant directory structure: `co-develop/`, `co-design/`, `co-work/`
- `validate-templates.ts` lifecycle validation script
- `templates/VERSION` semver tracking

### Changed
- All agent files updated with `## Meeting Participation` sections
- `## Dispatch Protocol` sections standardized across all agents
