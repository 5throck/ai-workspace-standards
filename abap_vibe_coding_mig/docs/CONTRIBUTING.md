## How to Contribute

### 1. Fork & Branch

```bash
git clone https://github.com/<your-fork>/abap.git
cd abap
git checkout -b feat/your-feature-name
```

### 2. Follow Project Conventions

- Read [CLAUDE.md](CLAUDE.md) for AI agent configuration
- Read [docs/context.md](docs/context.md) for codebase rules and ABAP development guidelines
- All documentation must be written in **English**
- ABAP scratch files go in `scratch/` only

### 3. Commit & Push

```bash
git commit -m "feat: describe your change clearly"
git push origin feat/your-feature-name
```

### 4. Open a Pull Request

- Describe **what** and **why** in the PR description
- Reference any related issues
- Wait for CLA bot confirmation and review
