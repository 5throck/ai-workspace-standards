## Decisions
- All chrome/UI surfaces use CSS variables defined per-style, not hardcoded rgba values
- Dark styles → dark glass (--glass-bg: rgba(17,24,39,0.92)), light styles → light glass (--glass-bg: rgba(255,255,255,0.92))

- Accent fallback unified: removed #2563eb (blue), all themes use var(--accent-color) which resolves from style.css
- pitch theme TOC uses same variable system as ppt-engine.css themes despite separate CSS
- All TOC containers standardized to <ul> since TOCBuilder creates <li> children
